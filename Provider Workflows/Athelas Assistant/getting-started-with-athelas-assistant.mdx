---
title: "Getting Started with Athelas Assistant"
---

Athelas Assistant is a powerful tool to provide users with the familiar ChatGPT-like help along with powerful agentic tools. For v1, we have implemented a few key features, such as answering medical questions, syncing with the current chart note, and pulling up patient or appointment data.

#### Usage

Pro tip: Use dictation or voice mode in the Assistant input box!

To start:

- Click the blue button in the bottom-right corner of the screen to show the AI Widget

<img src="/images/Provider%20Workflows/Athelas%20Assistant/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-1.png" alt="" />

- Click “Athelas AI” to open the Athelas AI widget, and then click on the “Assistant” tab. You’ll see a chat history page with no current chats, click on the “Start New Chat” button to start chatting

<img src="/images/Provider%20Workflows/Athelas%20Assistant/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-2.png" alt="" />

- You can also create a chat linked to the scribe from the completed scribe view - to ask questions with the full transcript and note as context

<img src="/images/Provider%20Workflows/Athelas%20Assistant/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-3.png" alt="" />

- You’ll see some quick start suggestions, but **also ask it directly what it can do!**
- You can always go back to your chat history to view older chats and create new chats
- Start fresh by clicking the dropdown in the chat page to create a new chat

<img src="/images/Provider%20Workflows/Athelas%20Assistant/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-4.png" alt="" />
