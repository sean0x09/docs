title: 'Getting Started with Athelas Assistant'
---

<p>Athelas Assistant is a powerful tool to provide users with the familiar ChatGPT-like help along with powerful agentic tools. For v1, we have implemented a few key features, such as answering medical questions, syncing with the current chart note, and pulling up patient or appointment data.</p><h4>Usage</h4><p>Pro tip: Use dictation or voice mode in the Assistant input box!</p><p>To start:</p><ol><li data-preset-tag="p"><p>Click the blue button in the bottom-right corner of the screen to show the AI Widget</p></li></ol><img alt="" src="/images/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-1.png" /><ol><li data-preset-tag="p"><p>Click “Athelas AI” to open the Athelas AI widget, and then click on the “Assistant” tab. You’ll see a chat history page with no current chats, click on the “Start New Chat” button to start chatting</p></li></ol><img alt="" src="/images/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-2.png" /><ol><li data-preset-tag="p"><p>You can also create a chat linked to the scribe from the completed scribe view - to ask questions with the full transcript and note as context</p></li></ol><img alt="" src="/images/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-3.png" /><ol><li data-preset-tag="p"><p>You’ll see some quick start suggestions, but <strong>also ask it directly what it can do!</strong></p></li></ol><ul><li data-preset-tag="p"><p>You can always go back to your chat history to view older chats and create new chats</p></li><li data-preset-tag="p"><p>Start fresh by clicking the dropdown in the chat page to create a new chat</p></li></ul><img alt="" src="/images/getting-started-with-athelas-assistant/getting-started-with-athelas-assistant-4.png" /><h4><br /></h4><h4><br /></h4>
