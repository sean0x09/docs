---
title: "Set up Custom Chart Note Templates"
---

To begin setting up custom templates, first locate the side navigation menu item titled "Utilities." Inside here, click on "Templates."

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-1.png" alt="" />

On this Templates page, you are able to search and filter for your different templates

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-2.png" alt="" />

To create a new template, click on “+Create New Template” on the top right corner.

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-3.png" alt="" />

You’ll be navigated to a new page. This is how you will configure intake and chart note templates. At the top, you can title your template then drag and drop components from the left hand side to the right to add questions/sections into your template.

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-4.png" alt="" />

Components are what you will be using to create your desired intake form. Drag and drop the components from the left to the right to add intake form fields. The different components available to you are:

**Single Choice:** Ask a question where only a single choice can be chosen as the answer. Click the + to make a new option.

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-5.png" alt="" />

**Short Answer**: Ask a question where a short answer can be typed in, this is intended for short responses

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-6.png" alt="" />

**Multiple Choice**: Ask a question where multiple options can be chosen as the answer. Click the + to make a new option.

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-7.png" alt="" />

**Paragraph Answer**: Ask a question where a paragraph answer can be typed in, this is intended for long responses

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-8.png" alt="" />

**Yes/No Question**: Ask a question where only yes or no can be chosen as the answer

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-9.png" alt="" />

**Number Answer**: Ask a question where only a number can be inputted as the answer

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-10.png" alt="" />

**Rating Question**: Ask a question where a rating from 1-5 is determined as the answer

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-11.png" alt="" />

**Table Answer**: Ask a question in a table format. Each line creates a new column, click the + button to create a new one

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-12.png" alt="" />

**Date Question**: Ask a question where only a date can be inputted as the answer

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-13.png" alt="" />

**Text Box**: This component is not available for EHR templates and only for patient intake.

Once you’ve created your form, select the type of template you want it saved as and click on “Save”. You will create EHR chart note templates and patient intake forms based on what you save it as.

<img src="/images/Provider%20Workflows/Chart%20Notes/set-up-custom-chart-note-templates/set-up-custom-chart-note-templates-14.png" alt="" />
