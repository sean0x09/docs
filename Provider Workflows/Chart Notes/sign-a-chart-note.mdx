title: 'Sign a Chart Note'
---

<p>This section will compile all the compliance checkmarks within the chart note for a quick overview. If any of them are grey, the information was not captured within the chart note. This is also where the user can select co-signers under “Additional Providers” and for the rendering provider to sign as well.</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-1.png" /><p>Ensure the “Fax note to referring provider” checkbox is clicked. Optionally, you can include a section requesting signature if desired.</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-2.png" /><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-3.png" /><p>To close the encounter and submit it as a claim, sign the note under “Notarize” and click “Submit”</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-4.png" /><p>To fax the completed chart note, click on “+Create Fax”. A side panel will open up where the faxing type can be selected.</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-5.png" /><p><br /></p><p><strong>There are 3 faxing types currently available:</strong></p><ul><li data-preset-tag="p"><p>Provider</p></li><li data-preset-tag="p"><p>Individual Fax Number</p></li><li data-preset-tag="p"><p>Imaging Request.</p></li></ul><p>Each presents in a different way.</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-6.png" /><p>The provider fax has a dropdown selection of all referring providers within Air. If the fax number is available for the provider, it will auto-populate.</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-7.png" /><p>Individual fax number requires the user to know and input the fax number they’d like to send the completed chart note to.</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-8.png" /><p>Imaging requests will fax out the chart note with an imaging request utilizing the details indicated within the section. This information includes:</p><ul><li data-preset-tag="p"><p>Diagnosis codes</p></li><li data-preset-tag="p"><p>Procedure codes</p></li><li data-preset-tag="p"><p>Rule Out Information</p></li><li data-preset-tag="p"><p>Result Medium</p></li><li data-preset-tag="p"><p>Attachments</p><ul><li data-preset-tag="p"><p>Chart Note</p></li><li data-preset-tag="p"><p>Patient attachments</p></li></ul></li><li data-preset-tag="p"><p>Imaging facility</p></li></ul><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-9.png" /><p>When a note is signed and submitted, if the patient has future appointments scheduled- you will have the option of updating all future appointments with the chart note you just completed. Clicking <em>Next</em> will then ask if you’d like to review the appointment in RCM/ Billing. If you’d like to do so, simply click on <em>View Encounter Details</em> or <em>View Claim Details</em>. If not, click <em>Complete</em>.</p><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-10.png" /><img alt="" src="/images/sign-a-chart-note/sign-a-chart-note-11.png" />
