#### Summary

This report is a list of all patients and their current outstanding balances.

####

#### Notes

- This report does not include “customers” created through the quick purchase flow.
- This report only includes charges that are still on the the patient’s balance. If a charge is canceled it will not be shown in this report.

####

#### Filters Supported

- None. All patients will show in this report.
