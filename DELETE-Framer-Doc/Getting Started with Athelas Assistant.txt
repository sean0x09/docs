Getting Started with Athelas Assistant

<p>Athelas Assistant is a powerful tool to provide users with the familiar ChatGPT-like help along with powerful agentic tools. For v1, we have implemented a few key features, such as answering medical questions, syncing with the current chart note, and pulling up patient or appointment data.</p><h4>Usage</h4><p>Pro tip: Use dictation or voice mode in the Assistant input box!</p><p>To start:</p><ol><li data-preset-tag="p"><p>Click the blue button in the bottom-right corner of the screen to show the AI Widget</p></li></ol><img alt="" src="https://framerusercontent.com/images/y4GDo9xcTqkc3c7mzboyvudjso.png"><ol><li data-preset-tag="p"><p>Click “Athelas AI” to open the Athelas AI widget, and then click on the “Assistant” tab. You’ll see a chat history page with no current chats, click on the “Start New Chat” button to start chatting</p></li></ol><img alt="" src="https://framerusercontent.com/images/bQnVn5EGSPqT1XcvGrKO8hNnpE.png"><ol><li data-preset-tag="p"><p>You can also create a chat linked to the scribe from the completed scribe view - to ask questions with the full transcript and note as context</p></li></ol><img alt="" src="https://framerusercontent.com/images/V6nGpjMQ4do7D1uR3Ma7JhBJzg8.png"><ol><li data-preset-tag="p"><p>You’ll see some quick start suggestions, but <strong>also ask it directly what it can do!</strong></p></li></ol><ul><li data-preset-tag="p"><p>You can always go back to your chat history to view older chats and create new chats</p></li><li data-preset-tag="p"><p>Start fresh by clicking the dropdown in the chat page to create a new chat</p></li></ul><img alt="" src="https://framerusercontent.com/images/O57kkQga9MGyXs74yK7A3cEoJTY.png"><h4><br></h4><h4><br></h4>
