---
title: "Profile Features Not Supported"
---

**Features In Development**

- Store histories in the patient profile (ex. Surgical, Social, Family, etc)
- Patient activity timeline tracking chronological order of care events (ex. appointments scheduled, chart note signed, orders placed, lab results received, etc)
- Contact list for storing contact profiles linked to patients (ex. Attorney, Case Manager, Guardian, etc)
**Features Not Supported**

- Multiple types of phone numbers and addresses (ex. home phone, cell phone, work address, home address)
- Categorize attachments into folders
- AI auto-tagging attachments
- Preview file before faxing
- See user logs and version history
