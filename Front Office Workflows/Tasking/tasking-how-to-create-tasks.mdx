---
title: 'How to create tasks'
---

<p>On the tasks page, you will see two tabs — one for tasks assigned <em>to </em>you (My Tasks), and one for tasks assigned <em>by</em> you.</p><img alt="" src="/images/how-to-create-tasks/how-to-create-tasks-1.png" /><p>For this demonstration, we’ll create a new task.</p><p>First, click <code>+ Task</code>.</p><img alt="" src="/images/how-to-create-tasks/how-to-create-tasks-2.png" /><p>In the popup, entitled ‘Create Task,’ enter a title and a clear, detailed description for the task.</p><img alt="" src="/images/how-to-create-tasks/how-to-create-tasks-3.png" /><p>When you choose the ‘Assignee,’ the menu will automatically populate with the names of your colleagues with Insights accounts. Begin typing to narrow results to find the staff member you want, then click their name to select them.</p><img alt="" src="/images/how-to-create-tasks/how-to-create-tasks-4.png" /><p>Set the priority level, status, and due date for the task, if applicable.</p><p>When you set a task type, you can either select one of the existing options or you can create a new task type. To create a new task type, simply type in the box and then select <code>+ CREATE</code>, as shown below.</p><img alt="" src="/images/how-to-create-tasks/how-to-create-tasks-5.png" /><p>You can also upload a file related to the task. Note that larger files will take longer to upload.</p><p>Once the task creation modal is filled out to your specifications, click <code>Create</code>.</p><p>Additionally, you can associate a task to a specific patient or group of patients. You can see each patient’s related tasks on their patient profile.</p><img alt="" src="/images/how-to-create-tasks/how-to-create-tasks-6.png" /><p><strong>Features In Development:</strong></p><ul><li data-preset-tag="p"><p>Automated task creation based on specific triggers</p></li><li data-preset-tag="p"><p>Ability to see all tasks at the clinic/business level</p></li><li data-preset-tag="p"><p>Ability to track analytics around task completion rate</p></li></ul>
