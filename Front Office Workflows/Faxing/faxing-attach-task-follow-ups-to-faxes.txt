Attach Task Follow-ups to Faxes


<p>Now you can attach task follow-ups directly to incoming or outgoing faxes. This makes it simple to keep track of next steps—right from your fax list</p><h4>What is it? </h4><p>A new <strong>Manage Tasks</strong> button appears on the right of each fax. Clicking it brings up a table of tasks associated with that fax.</p><p>All task drawers are pre-filled with the fax number (and patient, when available) so you don’t have to re-enter context.</p><iframe src="https://www.youtube.com/embed/oLiYaLBfqWQ?iv_load_policy=3&amp;rel=0&amp;modestbranding=1&amp;playsinline=1&amp;autoplay=0&amp;mute=1" data-thumbnail="Medium Quality" frameborder="0" allow="presentation; fullscreen; accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture"></iframe><h4>How to Use</h4><p><strong>1. Locate the Manage Tasks Button</strong></p><p>Navigate to the patient attachments page.</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-1.png"><p>Look for the <strong>Manage Tasks</strong> button to the right of your each row within the attachments table.</p><ul><li data-preset-tag="p"><p><strong>If the attachment has incomplete tasks:</strong></p><p>You’ll see a red notification icon on the button.</p></li></ul><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-2.png"><ul><li data-preset-tag="p"><p><strong>If the attachment has completed tasks:</strong></p><p>You’ll see a green notification icon on the button.</p></li></ul><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-3.png"><ul><li data-preset-tag="p"><p><strong>If the attachment has no associated tasks:</strong></p><p>You’ll see the button without any notification icon.</p></li></ul><p><strong>2. Open the Tasks Drawer</strong></p><p>Click on the <strong>Manage Tasks</strong> button to bring up a drawer with tasks for that fax.</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-4.png"><p><strong>View details</strong> of a task by clicking the down arrow on the leftmost column:</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-5.png"><p>The task type and description will be displayed under the current row:</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-6.png"><p><strong>Edit</strong> a task by clicking its pencil icon:</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-7.png"><p>The <strong>Edit Task</strong> drawer appears over the table to edit a single task:</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-8.png"><p><strong>Delete</strong> a task with the trash icon:</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-9.png"><p><strong>Update the status</strong> of a task by clicking on the checklist icon:</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-10.png"><p>You can mark a task as Done, In Progress, Not Started, or Archived.</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-11.png"><p><strong>Add </strong>a new task<strong> </strong>via the <strong>New Task</strong> button at the bottom left:</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-12.png"><p><strong>4. Create a New Task</strong></p><p>Select the <strong>New Task</strong> icon at the bottom left of the drawer.</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-13.png"><p>Title your task, assign it to someone in your team, pick a priority, due date, task type, etc. You’ll see that this task is automatically associated with the current fax.</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-14.png"><p>Click <strong>Create</strong> once all required fields are completed.</p><p><strong>5. View from the Main Tasks Page</strong></p><p>Navigate to <a href="https://insights.athelas.com/tasks">https://insights.athelas.com/tasks</a>. Select the <strong>My Tasks</strong> tab to see tasks assigned to you, or <strong>Assigned by me</strong> to see tasks you have assigned to others.</p><p>A new column to the right of the table will display the linked fax to each task, if any.</p><p>Clicking on the link will bring up the fax document in a new window.</p><img alt="" src="/images/attach-task-follow-ups-to-faxes/attach-task-follow-ups-to-faxes-15.png"><h4>FAQ's</h4><p><strong>Can I link the same task to both a patient and a fax?</strong></p><ul><li data-preset-tag="p"><p>Yes, if a fax is already linked to a patient, creating a task for it links it to the patient automatically. That means you can access it the patient’s task page as well. You can modify this behavior by adding or removing patients from the Create Task form.</p></li></ul><p><strong>Can I add a fax to a task from the </strong><a href="http://insights.athelas.com/tasks">Tasks page</a><strong>?</strong></p><ul><li data-preset-tag="p"><p>No, currently we do not support adding a fax to a task from the Tasks page. You will have to create it from the Faxing page. You can, however, view and manage tasks linked to faxes from the Tasks page.</p></li></ul>
