Insights Prior Authorization


<h3>At a Glance</h3><p>Prior authorization must be present in many circumstances in order for insurers to pay out claims. This guide details where to enter and update this information in Insights.</p><p><br></p><h4>What Is Prior Authorization?</h4><p><strong>Before providing treatment or prescribing medication, physicians need to obtain approval from an insurer</strong>. This tactic, used by insurance companies to control costs, is called prior authorization. Without prior authorization, a health plan may not pay for treatment or medication.</p><p>Emergency care doesn't need prior authorization.</p><h3>Here is the <a href="https://insights.athelas.com/prior-authorizations">Prior Authorizations page</a>.</h3><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-1.png"><p>You can access it via the header of the list of encounters on the <a href="https://insights.athelas.com/encounter-details">Encounter Details page</a>. Click the ‘Prior Authorizations’ button.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-2.png"><p><br></p><h4>Page Header Features</h4><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-3.png"><p>At the top of the Prior Authorizations page, you can…</p><ul><li data-preset-tag="p"><p><strong>Set filters</strong> to show you only certain authorizations. For example, you can filter for a particular patient or authorizations set to expire by a certain date.</p></li><li data-preset-tag="p"><p><strong>Download a CSV</strong> file so you can view the data in a spreadsheet.</p></li><li data-preset-tag="p"><p><strong>Create a new authorization</strong>.</p></li></ul><p>Just below the page header, you can click the following column headers to re-sort the list of authorizations in either ascending or descending order:</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-4.png"><ul><li data-preset-tag="p"><p>Total Visits/Units</p></li><li data-preset-tag="p"><p>Remaining Visits/Units</p></li><li data-preset-tag="p"><p>Effective Date</p></li><li data-preset-tag="p"><p>Expiry Date</p></li></ul><p><em>These walkthroughs focus on how to add or update prior auth through </em><a href="https://insights.athelas.com/prior-authorizations"><em><strong>this page only</strong></em></a><em>.</em></p><p><br></p><h4>How to Add a New Prior Authorization</h4><p>We’ll begin by clicking <code>Create Authorization</code>.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-3.png"><p>A window will appear with fields for you to fill out.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-6.png"><p><strong>Pro Tip</strong></p><p>To expedite your search in many fields throughout Insights, begin typing out the entity for which you’re searching and select it when it appears in the list.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-7.png"><p>Fill in the patient name, authorization type (pre-certification or referral), auth number, and effective and expiration dates.</p><p>You can choose to track prior auth by visit count or CPT code unit count. For this example, we’ll select visit count.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-8.png"><p><em>Add any notes that would be useful for colleagues looking for information about this authorization. You can select or create any tags you like, which can help keep your authorizations organized.</em></p><p>Click <code>Create</code> and you’re done! You can now see your new authorization in the table. It may help to filter for it if there are many other authorizations.</p><p><br></p><h4>How to Update Existing Prior Authorization</h4><p>Filter for the authorization you would like to update and click the pencil icon.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-9.png"><p>Make any adjustments you like to the authorization.</p><p>For this example, we’ll change the tracking type from visit count to CPT code and unit count. Then, we’ll click <code>Update</code>.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-10.png"><p>The new information is now displayed for that authorization.</p><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-11.png"><h4><br></h4><h4>Other Ways to Update Prior Authorization in Insights</h4><p>You can update prior auth in two other places in Insights:</p><ul><li data-preset-tag="p"><p>The ‘Prior Authorizations’ tab on the patient’s profile</p></li></ul><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-12.png"><ul><li data-preset-tag="p"><p>Editing the Service Lines section of the corresponding encounter on the <a href="https://insights.athelas.com/encounter-details">Encounter Details page</a></p></li></ul><img alt="" src="/images/insights-prior-authorization/insights-prior-authorization-13.png">
