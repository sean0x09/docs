#### Scheduling appointments through the Calendar page

The calendar will be where you will primarily schedule appointments for patients. There are several ways to create an appointment on this page. The first way is through the “Create New” button on the top right corner. A dropdown list of several options will be available to you.

#### Single Appointment Creation

Click on “Create Appointment” to make one appointment only. A side panel will pop open for the appointment details to be listed.

<img src="/images/Front%20Office%20Workflows/Appointments/how-to-schedule-an-appointment/how-to-schedule-an-appointment-3.png" alt="" />

Here you will be able to select/ fill out the following fields:

- Appointment Start and End Time
- Appointment Date
- Rendering Provider
- Referring Provider
- Supervising Provider
- Patient
- Case
- Appointment Type
- Facility
- Format (In person or Virtual)
- Insurance

- Insurance Priority
- Prior Authorization

- Appointment Notes
- Add customizable tags to label appointments
<!-- HTML preserved: <blockquote> -->❗**Note**: Clicking “Sync to case” will save the insurance information, priority and prior authorizations to be populated each time an appointment is made for that patient with the given case.

<!-- HTML preserved: </blockquote> -->Our enhanced EHR Insurance search functionality will allow your team to search for insurance names more efficiently. This ensures you can quickly locate the correct insurance details and reduce friction in the process. Within this section, you will also have the ability to create a new case, add a new prior authorization number and change the insurance priority.

#### Adding a New Case

To add a new case, click on the “Create New Case” button. Ensure you have a patient selected first, otherwise you will be unable to create a new case.

<img src="/images/Front%20Office%20Workflows/Appointments/how-to-schedule-an-appointment/how-to-schedule-an-appointment-5.png" alt="" />

Another side panel will open up with fields to input the case name and details such as:

- Insurance Priority

- Insurance priority will be auto-populated based off what was indicated in the case. There is the ability to override this temporarily or permanently in the appointment page.

- Rendering Provider
- Referring Provider
- Referral
- Attachments
- Case Notes

<img src="/images/Front%20Office%20Workflows/Appointments/how-to-schedule-an-appointment/how-to-schedule-an-appointment-6.png" alt="" />

#### Bulk Create Appointments

Bulk Appointment Scheduling allows you to set up multiple recurring appointments for a specific provider and facility

Select your preferred days, time, and start and end date. If appointments are available, you can select the appointment time across each day accordingly
