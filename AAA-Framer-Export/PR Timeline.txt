PR Timeline

<h3>At a Glance</h3><p>The <strong>PR Timeline</strong> chronologically displays every event related to PR and Payments for a particular Date of Service (DoS). It <strong>functions as a snapshot of PR for a given DoS, at any point in time.</strong> You can view when the appointment took place, when claims were submitted, all the way through to when balances were updated and payment was fully applied.</p><h2>üï∞Ô∏è&nbsp;Finding the Timeline</h2><p>From the <a href="https://insights.athelas.com/v2/patient-responsibility">Patient Responsibility</a> page, navigate to the Patient View tab and <strong>choose a patient.</strong></p><img alt="" src="https://framerusercontent.com/images/vAcwvaLxPqxBuwH4nCA0verTk.png"><p>By default you‚Äôll land on the <strong>Charges tab.</strong> From there, select a date of service from the list and click ‚Äò<strong>View Details</strong>.‚Äô</p><ul><li data-preset-tag="p"><p>You can also filter by Date of Service, Provider, or PR status (Outstanding, Paid, Cancelled).</p></li></ul><img alt="" src="https://framerusercontent.com/images/4QUIeUrFWBG3rpQD007qkJN3t50.png"><p>From there, find the <strong>Timeline tab</strong> partway down the page, and the timeline will appear!</p><img alt="" src="https://framerusercontent.com/images/FmOtuJKDx9SgvgC9Bgq9PW83Ec.png"><p><strong>Configuration:</strong> There are toggles for showing or hiding <strong>credits transactions</strong> as well as other <strong>hidden events</strong>. Some Events are hidden due to their relatively inconsequential nature. For example, if a partial refund is required, then a full refund may be issued and then credits will be used to make up the difference. This event will exist, but it will be hidden from the PR Timeline to make the data more intuitive and readable.</p><p><br></p><h2>Parts of the PR Timeline</h2><h3>Cards</h3><p>The PR timeline is broken down into cards, each of which represents an event in the PR journey.</p><img alt="" src="https://framerusercontent.com/images/J5WQn8lvfPn7P2sKlBJnojGxXQ.png"><h3>Date and Time</h3><p>Each card begins with the date and time that the event occurred.</p><img alt="" src="https://framerusercontent.com/images/GJbLV6iqoOH6WanDUK7mob5lqms.png"><h3>Card Type</h3><p>Next, what type of event the card represents is shown.
