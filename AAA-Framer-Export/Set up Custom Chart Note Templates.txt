Set up Custom Chart Note Templates

<p>To begin setting up custom templates, first locate the side navigation menu item titled "Utilities." Inside here, click on "Templates."</p><img alt="" src="https://framerusercontent.com/images/QpVvWpRGYVFpSDrlRju54o5UX4.png"><p>On this Templates page, you are able to search and filter for your different templates</p><img alt="" src="https://framerusercontent.com/images/dSKIQIrl2SD7gcZSFBrsYUd2anw.png"><p>To create a new template, click on “+Create New Template” on the top right corner.</p><img alt="" src="https://framerusercontent.com/images/4r09iq6KKzi8Js6ujcDY1aQns.png"><p>You’ll be navigated to a new page. This is how you will configure intake and chart note templates. At the top, you can title your template then drag and drop components from the left hand side to the right to add questions/sections into your template.</p><img alt="" src="https://framerusercontent.com/images/lCYcaOgnPdB1xdhqQ30fGEwrVX0.png"><p>Components are what you will be using to create your desired intake form. Drag and drop the components from the left to the right to add intake form fields. The different components available to you are:</p><p><strong>Single Choice:</strong> Ask a question where only a single choice can be chosen as the answer. Click the + to make a new option.</p><img alt="" src="https://framerusercontent.com/images/FEnNUIUH3nuLfgokpeWexgAw6g.png"><p><strong>Short Answer</strong>: Ask a question where a short answer can be typed in, this is intended for short responses</p><img alt="" src="https://framerusercontent.com/images/53TexwXj3uc2HQveIn1ZO6EK1I.png"><p><strong>Multiple Choice</strong>: Ask a question where multiple options can be chosen as the answer. Click the + to make a new option.</p><img alt="" src="https://framerusercontent.com/images/3pdTVySQUkrPw2rrlcy5pnFRW9c.png"><p><strong>Paragraph Answer</strong>: Ask a question where a paragraph answer can be typed in, this is intended for long responses</p><img alt="" src="https://framerusercontent.com/images/nlOFlXvLhW1wcuqMFmb36YoVsU.png"><p><strong>Yes/No Question</strong>: Ask a question where only yes or no can be chosen as the answer</p><img alt="" src="https://framerusercontent.com/images/JxrJtSsfLThuphLbZ3b5CmKXrM.png"><p><strong>Number Answer</strong>: Ask a question where only a number can be inputted as the answer</p><img alt="" src="https://framerusercontent.com/images/53TexwXj3uc2HQveIn1ZO6EK1I.png"><p><strong>Rating Question</strong>: Ask a question where a rating from 1-5 is determined as the answer</p><img alt="" src="https://framerusercontent.com/images/opT64tiXfLO14XwxcOicgPBRPhQ.png"><p><strong>Table Answer</strong>: Ask a question in a table format. Each line creates a new column, click the + button to create a new one</p><img alt="" src="https://framerusercontent.com/images/G8JA2HbDYENukvD0ta21d4Lpa8g.png"><p><strong>Date Question</strong>: Ask a question where only a date can be inputted as the answer</p><img alt="" src="https://framerusercontent.com/images/KprQSFf3SsfeSUHqBm3lCPoJEc.png"><p><strong>Text Box</strong>: This component is not available for EHR templates and only for patient intake.</p><p><br></p><p>Once you’ve created your form, select the type of template you want it saved as and click on “Save”.
